# 搜索

## 属性

| 属性名 | 说明 | 类型 | 默认/可选值 | 必填 |
|:--------|:---------:|:---------:|:---------:|--------:|
| v-model | 绑定值 |`object`| - | 否 |
| list | 搜索条件，详细见下表 | `Array` | `[]` | 是 |
| type | 展示类型 |`string`|`option`（默认） 、`expand` | 否 |

### list 详细

| 属性名 | 说明 | 类型 | 默认/可选值 | 必填 |
|:--------|:---------:|:---------:|:---------:|--------:|
| label | 查询条件名 | `string` | - | 是 |
| key | 条件对应字段，不可重复 |`string`| - | 是 |

```js
[
  {
    label:'名称',
    key:'name'
  },{
    label:'班级',
    key:'class'
  }
]
```

## 事件

| 事件名 | 说明 | Type |
|:--------|:---------:|--------:|
| change | 选中值发生变化时触发 | `(value: {key:string}) => void` |
