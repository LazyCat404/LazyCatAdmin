# Vue 3 + Typescript + Vite


### 启动

```
npm run dev
```

### 构建

```
npm run build
```

### UI 框架

- [element-plus](https://element-plus.org/#/zh-CN)


### 主要插件依赖

- typescript: ts 语法检查

- vite：打包工具

- sass：css预编译

- vite-plugin-style-import：按需导入 .css 文件

- splitpanes：窗格拆分/大小控制

- clipboard：复制面板

- vuex-persistedstate: vuex 持久化

- vuedraggable：拖拽
